# GlassOS-FS
The GlassOS file system navigator.

## Commands (from `help` command)
```
cd DIR                          Change directory
ls                              List contents of current directory
clear                           Clears screen
quit                            Quits the fs navigator
pwd                             Prints the working directory
mkdir DIR                       Creates a new directory
rmdir DIR                       Removes an empty directory
rm FILE                         Removes a file
rmall DIR                       Removes a directory and it's contents recursively
mv FROM_FILE TO_FILE            Renames a file
cp FILE NEW_FILE                Copies a file
cat FILE                        Displays the contents of a file
new FILE                        Creates an empty file
new FILE CONTENTS...            Creates a file with the arguments as the contents

```
